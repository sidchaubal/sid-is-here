I have been interested in iPhone applications since the iPhone was first launched.
Being a computer science graduate it was but natural for me to try this new technology :)
Here are some small excerpts of my work in developing iPhone apps:


GWU 4 RIDE App
Managing the 4-Ride service(GW University’s free taxi service) via this app.
iPhone App fetches current position of the user via GPS and sends this information to the Operator server (Operator screen front-end) wirelessly.
The operator then dispatches the Taxi according to the request received and sends confirmation to the iPhone wirelessly. Full Report

Presto Mobile (for JackBe Corporation)
Design and Implementation of JackBe Visualization Application in Objective C.
Visualizes multi-dimensional Data in the form of Bar-Chart /Pie-Chart and Grid
• Data for drawing is fetched via URL using XML
• Application compatible with iOS 4.0 and iPad environment
• Uses Google’s Open Source Core-Plot API



 


HawkIT (Soon in Appstore)
An iphone app that not only reminds you of your trivial ToDo but also monitors them via GPS.
It’s a location based Todo system that can alert you of doing something when you are in the targeted ToDo location.
